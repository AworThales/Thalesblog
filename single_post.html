The Cat and The Dog of Introversion

Thales Solomon Aug 24 · 10 min
Share

Web programming refers to the writing, markup and coding in web development, which includes web content, web client, and server scripting language and network security. The most common languages used for web programming are XML, HTML, Javascript, Perl 5, and PHP.

We programming is different from just programming, which requires inter-disciplinary knowledge on application area, client, and server scripting and database technology.

Web programmer and web developer is the same thing. It refers to someone who writes codes that powers a website or software.

CATEGORY OR TYPES OF WEB PROGRAMMING

Web programming is categorized into client-side (Frontend) and server-side (Backend) coding.

The client-side needs programming related to accessing data from users and providing information. Some of the tools used are HTML, CSS, Javascript.

Frontend is the side of the website or software that users see and interact with. Like highlight text, play videos, expand and minimize images, clickable items, forms and more.

The server-side needs programming related to data retrieval, security and performance. Some of these tools used are ASP, Lutos Notes, PHP, Java and MYSQL
